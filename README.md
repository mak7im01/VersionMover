# VersionMover - Перемещение номера версии

Аддон для PulseSync с гибкими настройками, который позволяет перемещать номер версии приложения и настраивать его внешний вид

## Описание

Аддон автоматически находит кнопку с номером версии (например, "5.86.0") и позволяет настроить её положение и внешний вид через удобный интерфейс настроек

## Возможности

- Включение/выключение перемещения версии
- Полное скрытие номера версии
- Настройка отступов от краёв экрана (слайдеры)
- Изменение цвета фона кнопки
- Настройка прозрачности кнопки (10-100%)
- Все настройки применяются в реальном времени
